  <h1>Python OOPS Assignment</h1>

  <p>
    This repository contains a set of beginner-friendly Python programs based on
    <strong>Object-Oriented Programming (OOP)</strong> and
    <strong>Exception Handling</strong>.
    It is designed to help freshers understand how to use classes, objects,
    constructors, loops, strings, lists, dictionaries, and error handling in Python.
  </p>

  <h2>Project Purpose</h2>
  <p>
    The main purpose of this project is to practice core Python concepts through
    simple but useful programs. Each program is written using a class so that the
    learner can understand how real-world Python applications are structured in an
    organized and reusable way.
  </p>

  <h2>Concepts Used</h2>
  <ul>
    <li><strong>Class</strong> – A blueprint used to create objects.</li>
    <li><strong>Object</strong> – An instance of a class used to access methods and data.</li>
    <li><strong>Constructor</strong> – A special method used to initialize data when an object is created.</li>
    <li><strong>Exception Handling</strong> – Used to catch and manage errors safely using <code>try</code> and <code>except</code>.</li>
  </ul>

  <h2>Programs Included</h2>

  <h3>1. Palindrome Number Checker</h3>
  <p>
    This program checks whether the given numbers are palindrome numbers or not.
    A palindrome number is a number that reads the same from left to right and
    right to left.
  </p>
  <p>
    <strong>Example:</strong> 101, 222, and 6 are palindrome numbers.
  </p>
  <p>
    <strong>Concepts learned from this program:</strong>
  </p>
  <ul>
    <li>How to work with numbers using loops</li>
    <li>How to reverse a number mathematically</li>
    <li>How to compare original and reversed values</li>
    <li>How to write logic inside a class method</li>
  </ul>

  <h3>2. Character Frequency Dictionary</h3>
  <p>
    This program finds the frequency of each character in a given string and stores
    the result in dictionary format.
  </p>
  <p>
    <strong>Example:</strong> For the string <em>"data science and ai"</em>, the program counts
    how many times each character appears.
  </p>
  <p>
    <strong>Concepts learned from this program:</strong>
  </p>
  <ul>
    <li>How to process a string character by character</li>
    <li>How to use dictionaries in Python</li>
    <li>How to count repeated values</li>
    <li>How to create frequency-based logic</li>
  </ul>

  <h3>3. Punctuation Remover</h3>
  <p>
    This program removes punctuation marks from a paragraph and returns only the clean text.
    It ignores symbols such as commas, full stops, and other punctuation marks.
  </p>
  <p>
    <strong>Example:</strong> A paragraph with punctuation marks is cleaned and converted into a plain sentence.
  </p>
  <p>
    <strong>Concepts learned from this program:</strong>
  </p>
  <ul>
    <li>How to filter characters from a string</li>
    <li>How to use conditions inside loops</li>
    <li>How to clean text data</li>
    <li>How text preprocessing works in real applications</li>
  </ul>

  <h3>4. May 2026 Calendar Application</h3>
  <p>
    This program is a small calendar application developed using OOP. It takes a date
    number from the user and prints the corresponding day name in May 2026.
  </p>
  <p>
    <strong>Concepts learned from this program:</strong>
  </p>
  <ul>
    <li>How to build a simple application using a class</li>
    <li>How to map dates with day names</li>
    <li>How to validate user input</li>
    <li>How to create a practical logic-based program</li>
  </ul>
